Overview
IARA Engine is a personal project I've developed to learn OpenGL and computer graphics fundamentals. As I progress through my graphics programming journey, this engine serves as my playground for implementing and experimenting with various rendering techniques.
What I've Learned
This project has helped me understand:

Core OpenGL concepts and the modern rendering pipeline
3D mathematics for computer graphics
Scene graph management and object transformation
Shader programming (GLSL)
Graphics optimization techniques

Current Features

3D Mesh Loading: Support for loading and rendering 3D models
Basic Lighting System: Point lights and sky lights
Shadow Mapping: Single cascade PCF shadows
Editor Interface: Simple gizmos for scene manipulation
Multisample Anti-Aliasing: For smoother rendering
Scene Hierarchy: Basic parent-child relationship between objects
Material System: Simple (non-PBR) material application
Performance Metrics: Frame rate and rendering statistics

VULKAN SDK required  
  
ALT + MBL             -- to rotate the camera inside the editor  
ALT + MouseWheelHold  -- to move the camera inside the editor  
  
SHIFT + MBL           -- to select entities  
CTRL + W              -- activate translation gizmo of the currently selected entity  
CTRL + E              -- activate rotation gizmo of the currently selected entity  
CTRL + R              -- activate scale gizmo of the currently selected entity  
CTRL + Q              -- no gizmo
